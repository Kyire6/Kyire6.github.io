Java8ж–°зү№жҖ§

еүҚиЁҖ

Java8 жңүдёҖдёӘйқһеёёжҳҫи‘—зҡ„зү№зӮ№пјҢе°ұжҳҜжҸҗдҫӣдәҶеҮҪж•°ејҸзј–зЁӢпјҢжң¬ж–Үд№ҹе°Ҷд»Ӣз»Қ Java8 зҡ„еҮ дёӘж–°зү№жҖ§жқҘе®һзҺ°еҮҪж•°ејҸзј–зЁӢгҖӮеӯҰдјҡиҝҷдәӣ API иғҪеӨҹзј–еҶҷеҮәз®ҖеҚ•гҖҒе№ІеҮҖгҖҒжҳ“иҜ»зҡ„д»Јз ҒпјҲе°Өе…¶жҳҜеҜ№йӣҶеҗҲзҡ„ж“ҚдҪңпјүгҖӮJava8 ж–°зү№жҖ§еҢ…жӢ¬пјҡ

  • Lambda иЎЁиҫҫејҸ
  • Stream API
  • жҺҘеҸЈж–°зү№жҖ§
    • еҮҪж•°ејҸжҺҘеҸЈпјҲ@FunctionalInterfaceпјү
    • й»ҳи®Өж–№жі•пјҲdefaultпјү
  • Optional API
  • ж–№жі•еј•з”Ё

дёҖгҖҒLambda иЎЁиҫҫејҸ

Lambda иЎЁиҫҫејҸпјҢд№ҹеҸҜз§°дёәй—ӯеҢ…пјҢжҳҜ Java8 еҸ‘еёғзҡ„дёҖдёӘж–°зү№жҖ§гҖӮ

Lambda е…Ғи®ёжҠҠеҮҪж•°дҪңдёәдёҖдёӘж–№жі•зҡ„еҸӮж•°пјҲе°ҶеҮҪж•°дј йҖ’еҲ°ж–№жі•дёӯпјүгҖӮ

дҪҝз”Ё Lambda иЎЁиҫҫејҸеҸҜд»ҘдҪҝд»Јз Ғжӣҙз®ҖжҙҒзҙ§еҮ‘гҖӮ

дёӢйқўжҳҜ Lambda зҡ„дё»иҰҒзү№еҫҒпјҡ

  • еҸҜйҖүзұ»еһӢеЈ°жҳҺпјҡдёҚйңҖиҰҒеЈ°жҳҺеҸӮж•°зұ»еһӢпјҢзј–иҜ‘еҷЁеҸҜд»Ҙз»ҹдёҖиҜҶеҲ«еҸӮж•°еҖј
  • еҸҜйҖүзҡ„еҸӮж•°еңҶжӢ¬еҸ·пјҡдёҖдёӘеҸӮж•°ж— йңҖе®ҡд№үеңҶжӢ¬еҸ·пјҢдҪҶеӨҡдёӘеҸӮж•°йңҖиҰҒе®ҡд№үеңҶжӢ¬еҸ·
  • еҸҜйҖүзҡ„еӨ§жӢ¬еҸ·пјҡеҰӮжһңдё»дҪ“еҢ…еҗ«дәҶдёҖдёӘиҜӯеҸҘпјҢе°ұдёҚйңҖиҰҒдҪҝз”ЁеӨ§жӢ¬еҸ·
  • еҸҜйҖүзҡ„иҝ”еӣһе…ій”®еӯ—пјҡеҰӮжһңдё»дҪ“еҸӘжңүдёҖдёӘиЎЁиҫҫејҸиҝ”еӣһеҖјеҲҷзј–иҜ‘еҷЁдјҡиҮӘеҠЁиҝ”еӣһеҖјпјҢеӨ§жӢ¬еҸ·йңҖиҰҒжҢҮе®ҡжҳҺиЎЁиҫҫејҸиҝ”еӣһдәҶдёҖдёӘж•°еҖј

Lambda иЎЁиҫҫејҸзҡ„з®ҖеҚ•дҫӢеӯҗпјҡ

еҸҳйҮҸдҪңз”Ёеҹҹ

еҰӮжһңдҪ жӣҫдҪҝз”ЁиҝҮеҢҝеҗҚеҶ…йғЁзұ»пјҢд№ҹи®ёдјҡйҒҮеҲ°иҝҷз§Қжғ…еҶөпјҡйңҖиҰҒеј•з”Ёе®ғжүҖеңЁж–№жі•дёӯзҡ„еҸҳйҮҸгҖӮиҝҷдёӘеҸҳйҮҸеҝ…йЎ»еЈ°жҳҺдёә final зұ»еһӢпјҢе°ҶеҸҳйҮҸеЈ°жҳҺдёә final пјҢж„Ҹе‘ізқҖдёҚиғҪдёәе…¶йҮҚеӨҚиөӢеҖјгҖӮJava8 иҷҪз„¶ж”ҫжқҫдәҶиҝҷдёҖйҷҗеҲ¶пјҢеҸҜд»Ҙеј•з”Ёйқһ final еҸҳйҮҸпјҢдҪҶжҳҜиҜҘеҸҳйҮҸйҡҗжҖ§дёә final еҸҳйҮҸгҖӮиҷҪз„¶ж— йңҖеЈ°жҳҺ final зұ»еһӢпјҢдҪҶеңЁ Lambda иЎЁиҫҫејҸдёӯпјҢиӢҘйҮҚеӨҚдёәе…¶иөӢеҖјпјҢзј–иҜ‘еҷЁе°ұдјҡжҠҘй”ҷгҖӮ

дәҢгҖҒStream API

StreamпјҲжөҒпјүжҳҜдёҖдёӘжқҘиҮӘж•°жҚ®жәҗзҡ„е…ғзҙ йҳҹеҲ—е№¶ж”ҜжҢҒиҒҡеҗҲж“ҚдҪң

  • е…ғзҙ жҳҜзү№е®ҡзұ»еһӢзҡ„еҜ№иұЎпјҢеҪўжҲҗдёҖдёӘйҳҹеҲ—гҖӮJava дёӯзҡ„ Stream е№¶дёҚдјҡеӯҳеӮЁе…ғзҙ пјҢиҖҢжҳҜжҢүйңҖи®Ўз®—
  • ж•°жҚ®жәҗ жөҒзҡ„жқҘжәҗгҖӮеҸҜд»ҘжҳҜйӣҶеҗҲпјҢж•°з»„пјҢIO channelпјҢдә§з”ҹеҷЁ generate зӯү
  • иҒҡеҗҲж“ҚдҪң зұ»дјј SQL дёҖж ·зҡ„ж“ҚдҪңпјҢжҜ”еҰӮ filterгҖҒmapгҖҒreduceгҖҒfindгҖҒmatchгҖҒsortedзӯү

е’Ңд»ҘеүҚзҡ„ Collection ж“ҚдҪңдёҚеҗҢпјҢStream ж“ҚдҪңиҝҳжңүдёӨдёӘеҹәзЎҖзҡ„зү№еҫҒпјҡ

  • pipeliningпјҡдёӯй—ҙж“ҚдҪңйғҪдјҡиҝ”еӣһжөҒеҜ№иұЎжң¬иә«пјҢиҝҷж ·еӨҡдёӘж“ҚдҪңеҸҜд»ҘдёІиҒ”жҲҗдёҖдёӘз®ЎйҒ“пјҢеҰӮеҗҢжөҒејҸйЈҺж јпјҲfluent styleпјүгҖӮиҝҷж ·еҒҡеҸҜд»ҘеҜ№ж“ҚдҪңиҝӣиЎҢдјҳеҢ–
  • еҶ…йғЁиҝӯд»Јпјҡд»ҘеүҚеҜ№йӣҶеҗҲйҒҚеҺҶйғҪжҳҜйҖҡиҝҮ Iterator жҲ–иҖ… for-each зҡ„ж–№ејҸпјҢжҳҫзӨәзҡ„еңЁйӣҶеҗҲеӨ–йғЁиҝӣиЎҢиҝӯд»ЈпјҢиҝҷеҸ«еҒҡеӨ–йғЁиҝӯд»ЈгҖӮStream жҸҗдҫӣдәҶеҶ…йғЁиҝӯд»Јзҡ„ж–№ејҸгҖӮ
